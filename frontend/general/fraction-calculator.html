사용 방법

1

첫 번째 분수 입력

첫 번째 분수의 분자(위 숫자)와 분모(아래 숫자)를 입력하세요.

2

연산 선택

덧셈, 뺄셈, 곱셈 또는 나눗셈을 선택하세요. 또는 단일 분수를 약분하세요.

3

결과 확인

계산 버튼을 눌러 결과를 확인하세요. 답은 자동으로 최소 항으로 약분됩니다.

분수 공식

덧셈

a/b + c/d = (a×d + b×c) / (b×d)
예제: 1/4 + 1/6 = (1×6 + 4×1) / (4×6) = 10/24 = 5/12

뺄셈

a/b - c/d = (a×d - b×c) / (b×d)
예제: 3/4 - 1/6 = (3×6 - 4×1) / (4×6) = 14/24 = 7/12

곱셈

a/b × c/d = (a×c) / (b×d)
예제: 2/3 × 3/4 = (2×3) / (3×4) = 6/12 = 1/2

나눗셈

a/b ÷ c/d = (a×d) / (b×c)
예제: 2/3 ÷ 3/4 = (2×4) / (3×3) = 8/9

실용 예제

레시피 배율 조정

원래: 2/3 컵 밀가루
2배 레시피: 2/3 × 2
결과: 4/3 = 1⅓ 컵

목재 자르기

판재: 8/4 피트 (2 피트)
3조각으로 자르기
결과: 8/4 ÷ 3 = 2/3 피트씩

페인트 혼합

빨강: 3/8 갤런
파랑: 1/4 갤런
총계: 3/8 + 2/8 = 5/8 갤런

숙제 문제

계산: 5/6 + 2/3
결과: 5/6 + 4/6 = 9/6 = 3/2 = 1½

자주 묻는 질문

대분수란 무엇인가요?

대분수는 정수와 분수를 결합한 것으로 2½와 같습니다 (가분수로는 5/2와 같습니다).

분수를 어떻게 약분하나요?

분자와 분모를 최대공약수(GCD)로 나눕니다. 예: 6/9 ÷ 3 = 2/3.

정수를 입력할 수 있나요?

네! 정수를 분자로 입력하고 1을 분모로 입력하세요. 예: 5 = 5/1.

가분수란 무엇인가요?

가분수는 분자가 분모보다 큰 분수로 7/4와 같습니다 (대분수로는 1¾와 같습니다).

분모가 0일 수 없는 이유는?

0으로 나누기는 수학에서 정의되지 않습니다. 분모가 0인 분수는 의미가 없습니다.

주요 기능

모든 연산

자동 약분 기능이 있는 분수 덧셈, 뺄셈, 곱셈, 나눗셈 및 약분.

자동 약분

변환
